int swd_drv_transmit ( swd_ctx_t *  swdctx,
swd_cmd_t *  cmd 
)

Transmit selected command from the command queue to the interface driver.

Also update the swdctx->log structure (this should be done only here!). Because commands that were queued does not get ack/parity data anymore, we need to verify ACK/PARITY that was just read and return error if necesary. When ACK/PARITY error is detected queue tail is removed as it is invalid.

Parameters:
*swdctxswd context pointer.
*cmdpointer to the command to be sent.
Returns:
number of commands transmitted (1), or SWD_ERROR_CODE on failure.
