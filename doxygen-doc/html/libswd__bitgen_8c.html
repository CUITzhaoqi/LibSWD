Generate 8-bit SWD-REQUEST packet contents with provided parameters.

Note that parity bit value is calculated automatically.

Parameters:
*swdctxswd context pointer.
*APnDPAccessPort (high) or DebugPort (low) access type pointer.
*RnWRead (high) or Write (low) operation type pointer.
*addrtarget register address value pointer.
*requestpointer where to store resulting packet.
Returns:
number of generated packets (1), or SWD_ERROR_CODE on failure.
